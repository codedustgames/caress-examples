# Sample projects using the [Caress Framework][]

Each folder is a standalone project that needs a copy or link to a copy of the
caress-framework in a folder called "caress". Then, each project can be run
from within it with:

```bash
love .
```

LOVE version 0.9.2 should be used.

[Caress Framework]: https://github.com/codedustgames/caress-framework
